# Shree Shyam Hast Rekha Gyan

A single-page static website for professional palmistry guidance services.

## Tech stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- Fully static export (`out/`), no backend or database
- SEO-friendly with semantic HTML and meta tags

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build (static export)

```bash
npm run build
```

Static files are in `out/`. Deploy `out/` to any static host (Vercel, Netlify, GitHub Pages, etc.).

## Content

Edit copy and links in **`src/config/content.ts`**. No CMS; all content is in that file.

## Design

- Light background (off-white / soft beige)
- Accent colors: deep green, muted gold, soft brown
- Mobile-first, responsive
- Minimal animations; subtle transitions only
